# Movie Info Trailer App

**Movie Info Trailer App** is a web application built with **HTML, CSS, and Python** that allows users to watch movie trailers and explore detailed movie information using a movie API.  

## Features
- Watch movie trailers directly in the app  
- View detailed movie info: plot, cast, ratings, release date, genre  
- Search movies and get quick results  
- Interactive and user-friendly interface  

## Tech Stack
- **Frontend:** HTML, CSS  
- **Backend:** Python  
- **API Integration:** Movie API with API Key
